# ConsultandoCEP

<https://consultandocep.netlify.com/>

## Visão Geral 

* Aplicação para treino 
* Construído com Javascript (Vanilla) e CSS puro, sem utilização de Frameworks
* Para consulta, é utilizado Fetch API consumindo da API do VIACEP
